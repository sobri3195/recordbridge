# Conflict Resolution Center

## Overview
Fitur **Conflict Resolution Center** adalah pusat pengelolaan konflik data yang berasal dari multiple sources. Fitur ini menyediakan interface untuk membandingkan data side-by-side, melacak histori keputusan, mengelola approval workflow, dan melakukan merge dengan audit trail yang lengkap.

---

## Jenis Konflik yang Ditangani

### 4.1 Conflict Categories

```typescript
interface ConflictDefinition {
  id: string;
  category: ConflictCategory;
  severity: 'CRITICAL' | 'HIGH' | 'MEDIUM' | 'LOW';
  status: 'OPEN' | 'UNDER_REVIEW' | 'RESOLVED' | 'ESCALATED';
  
  // Conflict Details
  entityType: 'PATIENT_DEMOGRAPHIC' | 'DIAGNOSIS' | 'MEDICATION' | 
              'ALLERGY' | 'VITAL_SIGN' | 'LAB_RESULT' | 'CONTACT_INFO';
  field: string;
  canonicalField: string;
  
  // Conflicting Values
  sources: Array<{
    sourceSystem: string;
    sourceRecordId: string;
    value: any;
    rawValue: any;
    timestamp: string;
    confidence: number;
    provenance: Provenance;
  }>;
  
  // Resolution
  resolution?: ConflictResolution;
  history: ConflictHistoryEntry[];
}

type ConflictCategory =
  | 'VALUE_MISMATCH'      // Same field, different values
  | 'SEMANTIC_DISAGREE'   // Different interpretations (e.g., NKA vs Penicillin)
  | 'TEMPORAL_GAP'        // Outdated vs current data
  | 'AUTHORITY_DISPUTE'   // Which source is more authoritative?
  | 'UNIT_INCOMPATIBLE'   // Different units (mg vs mcg)
  | 'STRUCTURAL_DIFF';    // Different data structures
```

### 4.2 Conflict Detection Matrix

| Scenario | Conflict Type | Example |
|----------|---------------|---------|
| BP 120/80 vs 118/82 | VALUE_MISMATCH | Minor variation, acceptable |
| Allergy: NKA vs Penicillin | SEMANTIC_DISAGREE | Critical contradiction |
| Med: Amlodipine 5mg vs 10mg | VALUE_MISMATCH | Dose discrepancy |
| Diagnosis: DM vs Pre-DM | SEMANTIC_DISAGREE | Clinical significance |
| Lab: HbA1c 6mo ago vs 1mo ago | TEMPORAL_GAP | Most recent wins? |
| Hospital A vs Clinic B record | AUTHORITY_DISPUTE | Which source to trust? |
